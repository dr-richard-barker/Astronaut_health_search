This study presents an integrated analysis of spaceflight-induced transcriptional changes and an "opposite-forcing" drug/flavonoid reversal screen using R and LINCS L1000 data. We identify candidate oncogenic biomarkers altered by spaceflight and propose compounds that reverse these signatures in vitro. Supporting RNA-seq analysis from the JAXA6 human-cell dataset is included.
